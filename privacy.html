Politique de confidentialité de Mon Bot Discord

1. Collecte de données

Ce bot ne collecte aucune donnée personnelle. Il n’enregistre pas vos messages, serveurs ou informations personnelles.

2. Données techniques

Le bot peut temporairement traiter des informations techniques (comme votre ID Discord) uniquement pour fonctionner correctement, mais il ne les stocke pas.

3. Partage de données

Aucune donnée n’est partagée avec des tiers.

4. Modifications

Cette politique de confidentialité peut être mise à jour. Les changements seront affichés sur cette page.

5. Contact

Pour toute question sur la confidentialité, contactez-nous à ton-email@example.com.
